# 爬墙车三代 RX 自动化测试仪 - 操作说明书

## 一、产品简介
本测试仪为爬墙车三代 RX 接收板专用自动化测试设备，可快速完成 RX 板核心功能检测，精准筛选不良品，提升生产良率。
支持**指示灯+蜂鸣器**直观判断（不接串口可直接生产测试），同时提供**串口日志**用于维修深度定位故障。

### 测试项目
1. 2.4G 无线配对与通信功能
2. 电源电压检测电路
3. 过流保护检测电路
4. 马达驱动输出电路
5. 整机休眠功能（电流＜10μA）

---

## 二、硬件接线说明

### 1. 供电与基础
- 测试仪供电：3脚 VDD 2.4V–3.6V
- 地线：2脚、15脚 GND
- VDD 与 GND 之间建议并联 10μF 滤波电容
- 天线：1脚、16脚
- 晶振：14脚

### 2. 马达检测接口（建议串10K电阻保护）
- 4脚 → RX M1+
- 5脚 → RX M1-
- 9脚 → RX M2+
- 10脚 → RX M2-
- 6脚 → RX M2+/M3-

### 3. 外设接口
- 7脚（TX）→ 串口工具 RXD（可选，维修使用）
- 8脚 → 蜂鸣器（8050驱动）
- 12脚 → 绿灯（低电平亮）
- 13脚 → 红灯（低电平亮）
- 11脚 → 测试按键（对地）

### 4. 重要要求
测试仪、RX 板、串口工具 **必须共地**

---

## 三、测试准备
1. RX 供电：4.2V（误差±0.05V）
2. 可串联电流表观察休眠电流
3. **测试开始前：不遮挡红外对管**
4. 上电顺序：先测试仪 → 后 RX

---

## 四、操作步骤（接串口 / 不接串口 都能用）

### 1. 测试仪上电
红绿指示灯交替闪烁
**上电无任何串口输出**

### 2. RX 上电
RX 橙灯呼吸闪烁 → 正常
无呼吸 → 直接判定不良

### 3. 启动测试（关键）
按下测试按键
**按下后立即遮挡红外对管 → 自动开始发码与测试**
**按下按键后，串口立即输出：Start Test check pair......**

### 4. 自动测试流程
配对 → 红外检测 → 马达负载 → 电压检测 → 过流检测 → 马达输出 → 休眠

---

## 五、结果判断

### ✅ 测试通过
- 绿灯常亮
- 蜂鸣器短响一声
- RX 橙灯熄灭，进入休眠
- 电流＜10μA
- 串口全部显示 OK

### ❌ 测试失败
- 红灯常亮
- 蜂鸣器长响
- 串口显示对应 Fail 信息（维修参考）

### 连续测试
换 RX → 按测试键 → 重新开始
无需断电

---

## 六、串口参数
- 波特率：76800
- 校验位：无
- 停止位：1

---

## 七、串口故障代码说明
- check VDD......VDD H......Fail → 分压偏高
- check VDD......VDD L......Fail → 分压偏低
- check VDD......Cap lost......Fail → 滤波电容异常
- check current......Cap1 Lost......Fail → 过流电容1异常
- check current......Cap2 Lost......Fail → 过流电容2异常
- check current......Res1 Lost......Fail → 过流电阻1异常
- check current......Res2 Lost......Fail → 过流电阻2异常
- check Motor......Motor1P Lost......Fail → M1+ 驱动异常
- check Motor......Motor1N Lost......Fail → M1- 驱动异常
- check Motor......Motor2P Lost......Fail → M2+ 驱动异常
- check Motor......Motor2N Lost......Fail → M2- 驱动异常
- check Motor......Motor3 Lost......Fail → M3 驱动异常
- check sleep.....Fail → 无法休眠

---

## 八、重要注意事项
1. 按测试键前：**不遮挡红外**
2. 按测试键后：**立即遮挡红外**（必须操作）
3. RX 必须使用 4.2V 电源
4. 所有设备共地
5. 不接串口可正常测试，接串口可查看详细故障
